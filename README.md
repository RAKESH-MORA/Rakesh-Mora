## 📌 Featured Projects

### 🧠 Classification of Pulmonary Diseases from Respiratory Sounds

Machine learning system that analyzes respiratory audio signals and classifies pulmonary diseases.

**Technologies Used**

- Python
- CNN (Convolutional Neural Networks)
- MFCC Feature Extraction
- NLP Techniques
- Flask Web Framework

**Project Overview**

This project focuses on detecting pulmonary diseases using respiratory sound recordings. The system extracts audio features using **MFCC (Mel-Frequency Cepstral Coefficients)** and processes them using a **Convolutional Neural Network (CNN)** model for classification.

A Flask-based web interface allows users to upload respiratory audio samples and receive predicted disease classifications.

**Key Components**

- Audio preprocessing and feature extraction
- MFCC signal representation
- Deep learning classification using CNN
- Flask web interface for user interaction
